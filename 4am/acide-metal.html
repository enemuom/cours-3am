تفاعل محلول حمضي مع معدن

⚗️ المادة وتحولها 📅 المقطع 2 — يناير ⏱ 60 دقيقة 📚 منهاج الجيل الثاني 2025/2026

🎯 الأهداف التعلمية

  • وصف تجربة تفاعل محلول حمض كلور الماء HCl مع معادن (Zn, Al, Fe)
  • الكشف عن نواتج التفاعل (غاز H₂ والشاردة المعدنية)
  • كتابة معادلة التفاعل الكيميائي الإجمالية والشاردية
  • تطبيق قواعد الأمن الكيميائي عند التعامل مع المحاليل الحمضية
🧪

I. وصف التجربة

🔬 تجربة تفاعل HCl مع الزنك Zn

نُضيف قطعة من معدن الزنك إلى محلول حمض كلور الماء (HCl). نلاحظ مباشرةً:

🫧 تصاعد فقاعات غاز من سطح المعدن — هذا الغاز هو غاز الهيدروجين H₂

🌡️ ارتفاع درجة حرارة المحلول — التفاعل طارد للحرارة

🔵 ذوبان المعدن تدريجياً — يتناقص حجم قطعة الزنك مع الزمن

🟦 تحوّل المحلول — يصبح محلولاً يحتوي على شوارد Zn²⁺

🔍 الكشف عن غاز الهيدروجين H₂

نقترب بعود ثقاب مشتعل من فوهة الأنبوب الذي يتصاعد منه الغاز:

📌 اختبار غاز H₂

غاز الهيدروجين يُنتج فرقعة خافتة عند اقتراب لهب منه. هذا الاحتراق يثبت وجود H₂:

2H₂ + O₂ → 2H₂O

⚗️

II. كتابة معادلة التفاعل

📝 تفاعل HCl مع الزنك Zn

محلول HCl يحتوي على: [H⁺ , Cl⁻]
الشوارد المتفاعلة هي H⁺ فقط (Cl⁻ لا تتفاعل مع الزنك — تبقى متفرجة):

المعادلة الشاردية:   Zn + 2H⁺ → Zn²⁺ + H₂↑
الزنك يتفاعل مع شوارد H⁺ فينتج شوارد Zn²⁺ وغاز H₂
المعادلة الإجمالية:   Zn + 2HCl → ZnCl₂ + H₂↑
التحقق: Zn(1=1)✓ | H(2=2)✓ | Cl(2=2)✓

📋 تفاعلات المعادن الأخرى مع HCl

المعدن المعادلة الشاردية المعادلة الإجمالية ملاحظة
الزنك Zn Zn + 2H⁺ → Zn²⁺ + H₂↑ Zn + 2HCl → ZnCl₂ + H₂↑ محلول عديم اللون
الألومنيوم Al 2Al + 6H⁺ → 2Al³⁺ + 3H₂↑ 2Al + 6HCl → 2AlCl₃ + 3H₂↑ تفاعل أسرع
الحديد Fe Fe + 2H⁺ → Fe²⁺ + H₂↑ Fe + 2HCl → FeCl₂ + H₂↑ محلول أخضر فاتح
الماغنيزيوم Mg Mg + 2H⁺ → Mg²⁺ + H₂↑ Mg + 2HCl → MgCl₂ + H₂↑ تفاعل قوي جداً

💡 القاعدة العامة: معدن + حمض كلور الماء → ملح + غاز هيدروجين
الشاردة H⁺ هي المسؤولة عن التفاعل في كل الأحماض المائية.

🔬

III. الاحتياطات الأمنية

⚠️ قواعد الأمن الكيميائي

⚠️ احتياطات إلزامية

🥽 ارتداء نظارات الحماية دائماً عند التجارب
🧤 ارتداء القفازات عند التعامل مع الأحماض
🌬️ إجراء التجارب في مكان جيد التهوية (H₂ غاز قابل للاشتعال)
🚫 عدم اقتراب لهب النار من الأحماض المركّزة
💧 عند إراقة حمض على الجلد: غسل فوري بكميات كبيرة من الماء
🔬 لا تُضاف المعادن النشيطة (Mg, Na) إلى الأحماض المركّزة

📊

IV. نشاط الأحماض — المقارنة

📈 نشاط المعادن مع الأحماض

ليست كل المعادن تتفاعل مع الأحماض بنفس الطريقة. بعض المعادن لا تتفاعل أصلاً:

المعدنالتفاعل مع HClالسبب
الماغنيزيوم Mg✅ تفاعل قوي جداًمعدن نشيط جداً
الألومنيوم Al✅ تفاعل قويمعدن نشيط
الزنك Zn✅ تفاعل متوسطمعدن نشيط نسبياً
الحديد Fe⚠️ تفاعل بطيءمعدن أقل نشاطاً
النحاس Cu❌ لا تفاعلمعدن غير نشيط مع HCl
الفضة Ag❌ لا تفاعلمعدن نبيل — لا يتفاعل

ملخص الكراس

📓 تفاعل محلول حمضي مع معدن — السنة الرابعة متوسط

  • 🧪
    التجربة: إضافة معدن (Zn, Al, Fe) إلى محلول HCl يُنتج:
    • تصاعد فقاعات غاز H₂ (فرقعة خافتة مع اللهب)
    • ذوبان المعدن تدريجياً
    • ارتفاع درجة حرارة المحلول
  • ⚗️
    تفاعل Zn مع HCl:
    شاردية: Zn + 2H⁺ → Zn²⁺ + H₂↑
    إجمالية: Zn + 2HCl → ZnCl₂ + H₂↑
  • ⚗️
    تفاعل Al مع HCl:
    شاردية: 2Al + 6H⁺ → 2Al³⁺ + 3H₂↑
    إجمالية: 2Al + 6HCl → 2AlCl₃ + 3H₂↑
  • ⚗️
    تفاعل Fe مع HCl:
    شاردية: Fe + 2H⁺ → Fe²⁺ + H₂↑
    إجمالية: Fe + 2HCl → FeCl₂ + H₂↑
  • 💡
    القاعدة العامة:
    معدن نشيط + HCl → ملح + H₂↑
    الشاردة H⁺ هي المسؤولة عن التفاعل — Cl⁻ متفرجة
  • ❌
    معادن لا تتفاعل مع HCl: النحاس Cu والفضة Ag (معادن غير نشيطة)
  • ⚠️
    الأمن الكيميائي: نظارات + قفازات + تهوية جيدة + إبعاد اللهب عن H₂
